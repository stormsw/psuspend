# psuspend
Process suspend/resume tool for win32 (built with VS2013 CE)

##Usage
```
psuspend <option> <pid1> <pid2> ...

Where optins are:
 Threads operations
-ts Threads Suspnde
-tr Threads reusme
-ns NT hack (undoc API) suspend
-nr NT hack (undoc API) resume
-ds Debugger start
-dr Debugger resume
```
